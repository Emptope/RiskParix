## 基础参数

1. 年化收益率
   计算：通过日收益率计算年化收益率，反映股票的长期回报。
2. 最大回撤
   计算：衡量股票从最高点到最低点的最大跌幅，反映其风险。
3. 波动率
   计算：通过日收益率的标准差衡量波动性，反映风险。
4. 夏普比率
   计算：衡量单位风险下的超额回报，评估风险调整后的收益。
